# Uncovering-Hidden-Risks-in-55M-Revenue-A-Sales-Performance-Investigation-Using-SQL-Power-BI
Uncovered hidden risks in $55.8M sales: zero customer growth, overreliance on 3 cities and 3 salespeople, plus major monthly volatility. Built an end-to-end SQL + Power BI dashboard (date dimension, star schema, DAX) and turned raw data into a clear story with 4 strategic recommendations.
